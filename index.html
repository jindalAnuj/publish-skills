🚀 publish-skills

Publish AI agent skills to Git repositories in one command

npm GitHub skills

⚡ Quick Start

npx publish-skills publish

That's it! Your skill is now a Pull Request away from being shared.

🎯 What It Does

publish-skills bridges the gap between creating AI agent skills and sharing them with the community. Combined with the skills package manager, it creates a complete publishing and distribution ecosystem.

🔄 How It Works

  1. Create your skill with npx publish-skills create
  2. Build your skill by editing SKILL.md and adding content
  3. Publish with npx publish-skills publish (creates a PR/MR)
  4. Share - once merged, anyone can install with npx skills add

  • GitLab - via @gitbeaker/rest
  • Bitbucket - via bitbucket API
  • Self-hosted Git - via Git CLI (Phase 2)

📦 What's a Skill?

A skill is a reusable package of instructions that extends your AI agent's capabilities:

my-awesome-skill/
├── SKILL.md           # Required: YAML frontmatter + instructions
├── README.md          # Optional: documentation
└── content/           # Optional: prompts, templates, resources
    ├── prompts/
    ├── templates/
    └── resources/

🛠️ Commands

  • npx publish-skills create - Create new skill from template
  • npx publish-skills validate [path] - Validate skill structure
  • npx publish-skills publish [path] - Publish skill (creates PR/MR)
  • npx publish-skills login/logout - Authenticate with Git platforms
  • npx publish-skills config - Manage configuration
